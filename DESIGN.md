# DESIGN.md

## 气质与意象
Facebook 直播电商的实时商品贴片工具——如同直播间里的智能导购牌，在主播演示商品的瞬间，屏幕上浮现优雅的价格标签和购买引导，让"看到即买到"成为直觉。

## 视觉策略
- 图形语言：圆角卡片 + 微投影，悬浮感强，与直播画面自然叠加
- 模板系统：10种预设模板覆盖 Ticker / Card / Badge / Banner / Countdown 五大类
- 每个模板有独立配色、动画节奏和布局逻辑

## 配色方案
- 主色：#1677FF（Ant Design 蓝，专业可信赖）
- 强调色：#FF4D4F（促销红，用于价格/限时标签）
- 成功色：#52C41A（已购买/完成状态）
- 中性灰：#F5F5F5 / #D9D9D9 / #8C8C8C（背景/边框/辅助文字）
- 暗色模式：直播画面背景默认深色，Overlay 使用高对比前景

## 字体排版
- 主字体：Inter / -apple-system（系统回退）
- 价格数字：Tabular Nums，粗体，大尺寸
- 中文：system-ui

## 动效与交互
- 商品入场：slideInLeft / slideInRight / fadeInUp
- 价格切换：数字翻转动画
- 高亮脉冲：1.2s ease-in-out infinite
- 倒计时：数字缩放 + 颜色渐变（绿→黄→红）
- 悬停：轻微上浮 + 阴影加深

## 页面结构
- Landing：功能展示 + CTA 注册
- Dashboard：左侧导航（商品/模板/Overlay/直播控制/数据分析/设置/账单）
- 直播控制：大号按钮 + 实时状态面板
- Overlay 预览：OBS 码一键复制

## 设计禁忌
- 不使用科技蓝+圆角卡片万能组合，需结合电商直播场景
- 禁止 Overlay 区域遮挡主播面部超过 20%
- 禁止在直播画面上使用纯白背景（反光刺眼）
- 禁止动画超过 3 层嵌套
